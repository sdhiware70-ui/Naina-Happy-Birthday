# **App Name**: Naina's Rx Birthday Bash

## Core Features:

- Birthday Announcement: Display a celebratory 'Stat!' message with medical-chic styling.
- Prescription Card: Show a stylized Rx card listing 'treatments' like '100% Joy'.
- Live Heartbeat Guestbook: Implement a Firestore-connected guestbook to display wishes in real-time, styled as 'Friendship Vital Signs'.
- Confetti Blast: Implement an 'Emergency' button that triggers confetti explosion and plays a birthday remix.
- Photo Gallery: Display a responsive CSS grid for sharing memorable photos.
- Firebase Integration: Connect the app to Firebase (Hosting, Firestore, Auth).

## Style Guidelines:

- Primary color: A muted lavender (#D0BFFF) to convey a sense of calmness and sophistication, aligning with a medical environment but keeping it gentle.
- Background color: Very light grey (#F5F5F5) to give the one-page design a clean and uncluttered feel.
- Accent color: A soft sky blue (#A0D2EB) to complement the lavender and add a touch of freshness, suitable for interactive elements.
- Body and headline font: 'PT Sans' sans-serif, suitable for headlines or body text
- Use heartbeat lines and pill-shaped elements as medical-themed design accents.
- Create a clean, one-page layout optimized for mobile responsiveness.
- Employ subtle animations for message updates and the confetti explosion.
